There is two different code in this folder. One of these is Bioinformatics Term Project.ipynb file. This is the main project code. To run this file one should follow steps given below:
1. Install python3
2. Pip3 install jupyter
3. Pip3 install numpy pandas matplotlib random sklearn scipy
4. Run 'jupyter notebook' command on the command line
5. Navigate the folder where the notebook in and open Bioinformatics Term Project.ipynb file.
The other code is the multilayer_bioinformatics.m file. This is the algorithm for implementing multilayer perception model and the results are printed in it. These is matlab file and one should run this code from Matlab.
